# Eleventy(11ty) Blog Template

## Features
- RSS Feed
- Minimalist Design
- Featured Posts
- A beautiful theme included out of the box
- Easily categorize posts with tags

## Installation
Clone the repo
```sh
git clone https://github.com/Mihir-27/11ty-Blog-Template.git
```
Install all pacakages
```sh
npm install
```
Run on localhost
```sh
npm run start
```
